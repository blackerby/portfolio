            Welcome

            Welcome to William Blackerby’s MLIS Portfolio #

            Photo by Quez Shipman

            This is William Blackerby’s portfolio of experiences from the online Master of Library and Information Studies program at the University of Alabama School of Library and Information Studies, created and submitted in partial fulfillment of the requirements for the MLIS degree.

            Use the menu on the left-hand side of the page to read my Philosophy of LIS Practice, reflections on SLIS’s program learning outcomes along with several work samples, and an up-to-date resume. I’ve also brought over some technical content from my blog which you can find under the Technology heading in the menu.

            Queries entered in the search bar at the top of the left-hand menu will return the names of documents containing your search term. Many of the documents on this site include a table of contents on the right-hand side of the page.

            This site was built using the Hugo static site generator with the Hugo Book Theme. It was deployed using GitHub Pages.
